# Data Structures Projects

A collection of Java implementations and coursework completed as part of CS 3305 - Data Structures.

This repository demonstrates core computer science concepts including linear data structures, trees, hashing, graph algorithms, recursion, and algorithm analysis.

## Technologies

* Java
* Object-Oriented Programming
* Data Structures
* Algorithm Design
* Big O Analysis

## Topics Covered

### Linked Lists

Implementation and manipulation of singly and/or doubly linked lists.

### Stacks and Queues

Implementation of stack and queue data structures and their applications.

### Sorting Algorithms

Analysis and implementation of common sorting algorithms with a focus on performance and algorithm complexity.

### Binary Search Trees

Tree traversal, insertion, deletion, and search operations using binary search trees.

### AVL Trees

Self-balancing binary search tree implementation and rotations.

### Hash Tables

Hashing techniques, collision handling, and efficient data retrieval.

### Graph Algorithms

Graph traversal and algorithmic problem solving using graph-based structures.

## Learning Outcomes

Through these projects I gained experience with:

* Object-oriented programming in Java
* Algorithm efficiency and Big O notation
* Recursive problem solving
* Tree and graph traversal techniques
* Data organization and retrieval methods
* Software debugging and testing
