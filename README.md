# Data Structures Projects

A collection of Java implementations and programming assignments completed as part of CS 3305 - Data Structures at Kennesaw State University.

This repository showcases the implementation of fundamental data structures and algorithms, emphasizing problem solving, object-oriented programming, recursion, data organization, and algorithmic efficiency.

## Technologies

- Java
- Object-Oriented Programming (OOP)
- Data Structures
- Algorithm Design
- Recursion
- Big O Analysis

## Repository Structure

### OOP and Recursion

Introduces object-oriented programming concepts and recursive problem solving through custom Java applications.

Topics explored:

- Classes and objects
- Encapsulation
- Recursive algorithms
- Problem decomposition

### Linked Lists

Implementation and manipulation of linked list data structures.

Topics explored:

- Dynamic memory organization
- Node traversal
- Insertion and deletion operations
- List management

### Stacks

Implementation of stack-based data structures and applications.

Projects include:

- Stack implementation
- Palindrome validation
- String reversal

Topics explored:

- LIFO data structures
- Stack operations
- Practical stack applications

### Queues & Radix Sort

Implementation of queue data structures and sorting algorithms.

Topics explored:

- FIFO data structures
- Queue operations
- Radix sort implementation
- Algorithm efficiency

### Heaps

Implementation of heap-based structures and priority queue concepts.

Projects include:

- Heap construction
- Heap sort
- Priority queue implementation

Topics explored:

- Tree-based storage
- Heap operations
- Efficient sorting techniques

### Binary Search Trees

Implementation of Binary Search Trees and tree-based algorithms.

Projects include:

- Tree traversal
- Searching
- Insertion and deletion
- Duplicate removal

Topics explored:

- Hierarchical data structures
- Recursive tree algorithms
- Efficient searching

### Hash Tables

Implementation of hashing concepts and hash functions.

Topics explored:

- Key-value storage
- Hash functions
- Collision handling
- Fast data retrieval

### Graphs

Implementation of graph-related algorithms and reachability analysis.

Topics explored:

- Graph representations
- Connectivity
- Reachability matrices
- Graph traversal concepts
